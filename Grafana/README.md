# Grafana Dashboard

This folder contains the Grafana dashboard used for real-time visualization of the **Edge AI-Based Predictive Maintenance System**.

The dashboard provides live monitoring of motor health by displaying temperature, vibration, AI anomaly score, and overall motor status received from the ESP32 through MQTT, Node-RED, and InfluxDB.

---

## Overview

The Grafana dashboard retrieves telemetry data stored in InfluxDB and presents it through interactive panels, allowing users to monitor the motor's condition in real time.

---

## Dashboard Architecture

```
ESP32
   │
   ▼
MQTT Broker (Mosquitto)
   │
   ▼
Node-RED
   │
   ▼

   │
   ▼

```

## Dashboard Features

The dashboard includes the following panels:

| Panel | Description |
|--------|-------------|
| **Motor Health** | Displays the current motor status (Normal / Warning). |
| **Current Temperature** | Shows the latest motor temperature in °C. |
| **Current Vibration** | Displays the latest vibration magnitude. |
| **Current AI Score** | Shows the latest anomaly score generated by the Autoencoder. |
| **Temperature Trend** | Live graph of motor temperature over time. |
| **Vibration Trend** | Live graph of vibration readings over time. |
| **Average Anomaly Score** | Displays the moving average of the AI anomaly score. |
| **Motor Health Gauge** | Visual gauge indicating overall motor condition. |

---

## Data Source

The dashboard is connected to **InfluxDB**, where all telemetry data is stored by Node-RED.

The following measurements are visualized:

| Measurement | Description |
|-------------|-------------|
| Temperature | Motor temperature |
| Vibration | Motor vibration magnitude |
| Raw AI Score | Reconstruction error generated by the Autoencoder |
| Average AI Score | Sliding window average anomaly score |
| Motor Status | Binary health indicator (0 = Normal, 1 = Anomaly) |

---

## Importing the Dashboard

1. Open Grafana.
2. Navigate to **Dashboards → Import**.
3. Select **dashboard.json**.
4. Choose the configured InfluxDB data source.
5. Click **Import**.

The dashboard will be ready for real-time monitoring.

---

## Auto Refresh

The dashboard is configured for live monitoring and automatically refreshes at regular intervals to display the latest sensor readings and AI predictions.

## Purpose

The Grafana dashboard provides an intuitive interface for monitoring motor health in real time by displaying:

- Live temperature monitoring
- Live vibration monitoring
- AI-based anomaly score
- Historical sensor trends
- Overall motor health status

This enables operators to detect abnormal machine behavior early and perform predictive maintenance before a failure occurs.

## Project

**Edge AI-Based Predictive Maintenance using ESP32 and TensorFlow Lite Micro**

This project combines Embedded Systems, TinyML, IoT, MQTT communication, Node-RED, InfluxDB, and Grafana to create a low-cost, real-time predictive maintenance solution for rotating machinery.
